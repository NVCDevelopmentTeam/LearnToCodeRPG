- Dịch tất cả các file *.rpy trong thu mục Vietnamese, đảm bảo 100% tiếng Việt và giữ nguyên comment code của file gốc để người ta biết được là mình đang dịch cái gì, chỉ dịch string sang tiếng Việt
- File nào hay phần nào đã dịch rồi thì bỏ qua không dịch nữa, chỉ dịch những phần chưa dịch mà thôi
- Đảm bảo sử dụng đúng văn phong của game learn to code rpg và từ ngữ đúng với game VIệt và thuần Việt, đúng với ngữ cảnh trong game
- Từ ngữ gần gũi với người Việt và dịch sao cho khớp với nội dung và cốt chuyện của game
